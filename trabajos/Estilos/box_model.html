box-sizing , border-box

el modelo de caja de un elemento es la forma en que se calcula el tamaño total del elemento, que incluye su ancho, altura, borde, relleno y margen. Por defecto, el modelo de caja de un elemento en CSS es el modelo de caja de contenido (content-box), lo que significa que el tamaño total del elemento solo incluye su ancho y altura, y los bordes, rellenos y márgenes se agregan al tamaño total del elemento.

a veces es más útil utilizar el modelo de caja de borde (border-box), que significa que el tamaño total del elemento incluye el ancho, altura, borde y relleno, pero no la margen. Esto permite controlar el tamaño total del elemento de una manera más predecible y consistente.

Para aplicar el modelo de caja de borde a un elemento en CSS, se utiliza la propiedad box-sizing con el valor border-box
